# Contributing

We welcome contributions to our project! Please follow these steps to contribute:

1. Fork the repository on GitHub.

2. Clone your forked repository to your local machine:

   ```
   git clone https://github.com/Aniket-508/heroicons-animated.git
   ```

3. Navigate to the project directory:

   ```
   cd heroicons-animated
   ```

4. Create a new branch for your feature or bug fix:

   ```
   git checkout -b your-branch-name
   ```

5. Install the project dependencies:

   ```
   pnpm install
   ```

6. After implementing your changes, build the project to check for errors:

```
pnpm build
```

7. Test the application to ensure your changes work as expected:

   ```
   pnpm lint
   ```

8. Commit your changes:

   ```
   git commit -m "Add [animation-type] animation to [icon-name]"
   ```

9. Push your changes to your fork:

   ```
   git push origin your-branch-name
   ```

10. Open a pull request on the original repository with a clear description of the animation you've implemented.

Thank you for contributing to our project!
